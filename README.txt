==============
Arduino Puffer
==============

Code to make the arduino_mfc_relay_controller into a puffer,
controlled by a switch to trigger a puff and a potentiometer to set
puff duration or over USB.

firmware
========

Contains the Arduino firmware files. See firmware/README.txt

host
====

Contains the code for controlling with a host computer.

python
------

See host/python/README.txt
